营销权益组件库 — 400+ 权益 React 组件库
架构设计 + 核心开发 · React · TypeScript · Rollup · Changesets
背景

营销平台管理 400+ 种营销权益（优惠券、折扣、积分、立减等），权益类型持续增长，原有方案每新增一种权益需硬编码。

结果
覆盖 8 个业务项目，接入 400+ 种权益，新权益接入 2 天→2 小时，零 breaking change 运行 18 个月+。
