### Schema 驱动营销中台 — BFF 动态生成 + 前端动态渲染

*前端架构 + BFF 架构设计 · React · TypeScript · Node.js · Serverless · Lerna + Nx*

国际化营销中台覆盖 **10+ 国家、20+ 种活动类型**，每种活动有不同字段组合和联动规则。设计 Schema 驱动架构，BFF 动态生成 → 前端动态渲染，新增活动类型前端零代码。

- **BFF 模板引擎**：模板驱动 Schema 生成，字段按模板分布式管理，联动模板从渲染模板提取 handler 复用，保证联动与初始渲染结构一致

- **前端渲染引擎**：嵌套路径拍平为 dot 路径，注册表支持 **35+** 组件动态渲染，发布订阅模式执行联动
- **Monorepo 治理**：Lerna + Nx + Yarn workspace 管理 **15+** 业务模块，独立版本 + 构建缓存 + 依赖治理
