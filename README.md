## 快速开始

### 今天该干什么？

打开 [面试方案/备战日历.md](面试方案/备战日历.md)，看当天计划。

### 面试当天看什么？

打开 [面试题库/急救包.md](面试题库/急救包.md)，只看核心句。

### 某个题目不确定？

打开 [面试题库/题库详解.md](面试题库/题库详解.md)，查对应题目。

### 要做 Mock 练习？

1. 搭一个 ChatGPT Project，上传本仓库核心文件
2. 把 [面试训练/master-prompt.md](面试训练/master-prompt.md) 设为项目指令
3. 输入 `开始 mock：字节` 开始练习
