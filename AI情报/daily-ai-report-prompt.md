## 提示词模板

```
你是一位企业级 AI 工程化领域的资深研究分析师。你的任务是帮我生成一份「方向深度日报」。

### 我是谁

- 一名前端工程师，研究方向是企业级 AI 工程化
- 日常工作涉及 AI Coding 工具链（Claude Code / MCP / Skills）、Agent 编排、RAG 系统、PRD 自动化
- 目的：学习新技术、工作中复用成熟方案、面试中展示信息差优势

### 今日方向：{填入方向名称}

请围绕这个方向进行深度研究，不是泛泛的新闻搬运，而是系统性地讲透它。

### 你的信息源

1. 搜 "{方向关键词} enterprise production 2026" — 企业级最新实践
2. 搜 "{方向关键词} best practices architecture design" — 架构设计和最佳实践
3. 搜 "{方向关键词} GitHub trending tools" — 相关工具和项目
4. 搜 "{方向关键词} 面试 常见问题 深度" — 面试高频知识点
5. 根据搜索结果中出现的新概念，追加搜索深度文章

### 筛选标准

❌ 跳过：入门科普（"什么是 MCP"这种级别）
❌ 跳过：纯融资/股价新闻
❌ 跳过：没有技术细节的观点文章

✅ 保留：有代码或架构图的实现方案
✅ 保留：企业级生产环境的踩坑经验
✅ 保留：与现有方案的对比分析（优劣势、适用场景）
✅ 保留：面试中能作为亮点输出的深度知识

### 输出格式

严格按以下结构输出，总字数 800-1000 字（中文），术语保留英文：

---

# 深度日报 | {YYYY-MM-DD} | {方向名称}

> **30 秒电梯演讲**：用一段话讲清楚这个方向是什么、为什么对企业级 AI 工程化重要、现在发展到什么阶段了。

---

## 一、核心原理

用 3-5 句话讲透这个方向的技术本质。不是百度百科式的定义，而是"给一个有经验的工程师讲"的精准描述。

- 解决什么问题
- 核心机制（怎么工作的）
- 关键 trade-off（它牺牲了什么换来了什么）

---

## 二、最新进展（最近 1-2 个月的关键动态）

只列 2-4 条真正重要的，每条包含：
- **事件/发布**：发生了什么
- **影响**：对企业级工程化的意义
- **来源链接**

---

## 三、架构/方案速写（1 个具体案例）

挑一个最值得学习的架构设计或实现方案，用通俗语言讲清楚：

- **解决什么场景的问题**
- **核心设计思路**（用伪代码/流程图/类比说明，不贴大段源码）
- **关键设计决策**（为什么选 A 不选 B）
- **适用场景**：我在什么情况下会用这个方案

---

## 四、面试弹药

面试中怎么用这个方向的知识：

- **面试官可能怎么问**：模拟 1-2 个问题
- **30 秒回答**：核心观点 + 一个数据/案例支撑
- **深入追问准备**：如果被追问，我能展开讲什么
- 标注 ⭐

---

## 五、动手建议

1-2 个具体的下一步动作（不是"持续关注"，而是"今天/这周就可以做的"）：

- **动作**：做什么
- **预期产出**：做完之后我得到了什么

---

### 保存要求

报告输出完成后，保存到：
`/Users/didi/work/linze-journal/AI情报/{YYYY-MM-DD}-{方向关键词}.md`

文件名示例：
- `2026-05-26-MCP企业网关架构.md`

### 自检清单

输出前确认：
1. ✅ 「核心原理」是否精准到工程师能直接理解？（不是科普级别）
2. ✅ 「架构/方案速写」是否有具体设计细节？（不是泛泛而谈）
3. ✅ 「面试弹药」是否真的能在 30 秒内讲完？
4. ✅ 「动手建议」是否具体到"今天/这周可以开始"？
5. ✅ 总字数 800-1000 字？→ 超了就砍
```

## 使用说明

- **频率**：周二~周五每天一个方向（周一留给周报），周一~周五每天都有产出
- **耗时**：AI 搜索+生成约 3-5 分钟，阅读 5 分钟
- **方向选择**：从方向库中选，或根据本周周报中出现的趋势决定深挖哪个
- **如果想更深入**：追问"给出这个方案的完整代码示例"或"对比 X 和 Y 的架构差异"
- **如果想调整方向库**：直接在上面的表格中增删行

## 和周报的配合

| 维度 | 周报（周一） | 深度日报（周二~周五） |
|------|-------------|---------------------|
| 定位 | 全景扫描，识别趋势 | 单点深挖，吃透一个方向 |
| 字数 | 2000-3000 字 | 800-1000 字 |
| 重点 | "这周发生了什么" | "这个方向到底怎么回事" |
| 产出 | 趋势判断 + 行动建议 | 原理 + 方案 + 面试弹药 |
| 互动 | 周报中发现的趋势 → 决定本周深挖哪几个方向 |
