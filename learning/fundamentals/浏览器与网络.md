## 学习路径

1. **渲染管线**：HTML→DOM→CSSOM→Render Tree→Layout→Paint→Composite
2. **性能影响**：Reflow（最贵）> Repaint > Composite（最轻）
3. **缓存策略**：HTML no-cache / 静态资源 max-age / API no-store
4. **HTTPS**：四步握手 → TLS 1.3 简化为 1-RTT
5. **事件循环**：同步→微任务→宏任务，每轮循环的执行顺序
6. **安全**：XSS（注入JS）/ CSRF（冒充请求）/ CSP（限制加载）
