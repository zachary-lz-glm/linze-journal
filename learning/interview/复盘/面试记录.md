### 算法题

**题目**：Move Zeroes（LeetCode 283 Easy）— 将数组所有 0 移到末尾，保持非零元素相对顺序
- 思路：先写了 O(n²) splice+push，然后尝试双指针
- 卡点：双指针实现有 off-by-one 错误、i+1 越界、面试官两次帮忙指出 bug
- 是否完成：❌ 未完成，最终代码仍有 bug
- 复杂度：应做到 O(n) 时间 O(1) 空间
- 正确解法：
```js
function moveZeroes(nums) {
  let k = 0;
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] !== 0) {
      [nums[k], nums[i]] = [nums[i], nums[k]];
      k++;
    }
  }
}
```
