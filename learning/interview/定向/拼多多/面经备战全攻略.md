## 一、面试流程与风格

### 标准流程（2-3轮技术 + HR）

| 轮次 | 时长 | 核心侧重 |
|------|------|---------|
| **技术一面** | 50-60min | 八股文 + 项目介绍 + 手写题 4+ 道 + 1-2 道算法 |
| **技术二面** | 50-60min | 场景设计题 + 项目深挖 + 算法题 |
| **技术三面**（部分） | 45-60min | 交叉面，综合深度，可能要求实现完整功能（如手写 Redux） |
| **HR面** | 15-30min | 职业规划、跳槽动机、薪资期望 |

### 关键特征

1. **先撕代码再聊项目** — 代码能力是硬门槛，做不出来可能直接挂
2. **手撕代码量全大厂最大** — 一场面试 4+ 道手写题，被称为"手撕最多、八股最多的公司"
3. **八股问得超细** — 稍微卡壳就追问下一层
4. **偏爱 JS 语言特性手写** — 不是 LeetCode 算法为主，而是 Promise/class 继承/编译函数等
5. **二面场景题占比大** — 需要真实设计思维，不能靠背
6. **面试环境可能很简陋** — 可能用纯文本编辑器（非 IDE），类似记事本手搓

### 时间线参考

投递 → 一面(1-2周内) → 二面(1-3天内) → HR面(1-3天内) → 录用意向(约1周内)

## 三、手撕代码（拼多多核心重点）

> **这是拼多多和 LeetCode 算法面试最大的区别**：拼多多前端最爱考 JS 语言特性手写，不是数据结构算法。

### 第一梯队：拼多多必考（极高频率）

**1. Promise 串行执行 + 重试机制**

```ts
// 串行执行
async function serialize(tasks: (() => Promise<any>)[]): Promise<any[]> {
  const results = [];
  for (const task of tasks) {
    results.push(await task());
  }
  return results;
}

// 带重试
function retry(fn: () => Promise<any>, times: number, delay: number): Promise<any> {
  return new Promise((resolve, reject) => {
    const attempt = (n: number) => {
      fn().then(resolve).catch(err => {
        if (n <= 1) reject(err);
        else setTimeout(() => attempt(n - 1), delay);
      });
    };
    attempt(times);
  });
}
```

**2. 模板编译函数（`{{}}` 替换）**

```ts
function compile(template: string, data: Record<string, any>): string {
  return template.replace(/\{\{(\w+(?:\.\w+)*)\}\}/g, (_, path) => {
    return path.split('.').reduce((obj, key) => obj?.[key], data) ?? '';
  });
}
// compile('{{user.name}} is {{age}}', {user: {name: 'Tom'}, age: 25})
// → 'Tom is 25'
```

**3. ES6 class 转 ES5（寄生组合继承）**

```ts
function _inherits(Child, Parent) {
  // 原型链继承
  Child.prototype = Object.create(Parent.prototype);
  Child.prototype.constructor = Child;
  // 静态方法继承
  Object.setPrototypeOf(Child, Parent);
}

function Parent(name) { this.name = name; }
Parent.prototype.say = function() { return this.name; };

function Child(name, age) {
  Parent.call(this, name); // 借用构造函数
  this.age = age;
}
_inherits(Child, Parent);
```

**4. 异步调度器（带并发限制）**

```ts
class Scheduler {
  private queue: (() => Promise<any>)[] = [];
  private running = 0;
  constructor(private max: number) {}

  add(task: () => Promise<any>): Promise<any> {
    return new Promise((resolve, reject) => {
      const run = () => {
        if (this.running >= this.max) return;
        this.running++;
        task().then(resolve).catch(reject).finally(() => {
          this.running--;
          if (this.queue.length) this.queue.shift()!();
        });
      };
      if (this.running < this.max) run();
      else this.queue.push(run);
    });
  }
}
```

**5. 发布订阅模式（EventEmitter）**

```ts
class EventEmitter {
  private events = new Map<string, Function[]>();

  on(event: string, fn: Function) {
    const list = this.events.get(event) || [];
    list.push(fn);
    this.events.set(event, list);
  }

  off(event: string, fn: Function) {
    const list = this.events.get(event);
    if (list) this.events.set(event, list.filter(f => f !== fn));
  }

  emit(event: string, ...args: any[]) {
    (this.events.get(event) || []).forEach(fn => fn(...args));
  }

  once(event: string, fn: Function) {
    const wrapper = (...args: any[]) => { fn(...args); this.off(event, wrapper); };
    this.on(event, wrapper);
  }
}
```

**6. 深拷贝（含循环引用）**

```ts
function deepClone<T>(obj: T, seen = new WeakMap()): T {
  if (obj === null || typeof obj !== 'object') return obj;
  if (seen.has(obj as any)) return seen.get(obj as any);

  if (obj instanceof Date) return new Date(obj) as any;
  if (obj instanceof RegExp) return new RegExp(obj) as any;

  const clone = Array.isArray(obj) ? [] : {} as any;
  seen.set(obj as any, clone);

  for (const key of Object.keys(obj as any)) {
    clone[key] = deepClone((obj as any)[key], seen);
  }
  return clone;
}
```

**7. 防抖 / 节流**

```ts
function debounce(fn: Function, delay: number) {
  let timer: any;
  return function(this: any, ...args: any[]) {
    clearTimeout(timer);
    timer = setTimeout(() => fn.apply(this, args), delay);
  };
}

function throttle(fn: Function, interval: number) {
  let last = 0;
  return function(this: any, ...args: any[]) {
    const now = Date.now();
    if (now - last >= interval) {
      last = now;
      fn.apply(this, args);
    }
  };
}
```

**8. 手写 call / apply / bind**

```ts
Function.prototype.myCall = function(ctx: any, ...args: any[]) {
  const fn = Symbol();
  ctx[fn] = this;
  const result = ctx[fn](...args);
  delete ctx[fn];
  return result;
};

Function.prototype.myBind = function(ctx: any, ...args1: any[]) {
  const fn = this;
  const bound = function(...args2: any[]) {
    return fn.apply(this instanceof bound ? this : ctx, [...args1, ...args2]);
  };
  bound.prototype = Object.create(fn.prototype);
  return bound;
};
```

### 第二梯队：大厂通吃

| 手写题 | 核心思路 |
|--------|---------|
| Promise.all | 计数器 + results 数组，全部 resolve 才 resolve |
| Promise.race | 谁先完成就 resolve/reject |
| Promise 超时控制 | `Promise.race([task, timeoutReject])` |
| instanceof | 沿 `__proto__` 链查找 prototype |
| new | 创建对象 → 原型绑定 → 执行构造函数 → 返回 |
| 数组扁平化 | 递归 reduce + concat，或迭代 + 栈 |
| URL 参数解析 | `new URLSearchParams()` 或 split 处理 |
| 对象列表转树 | Map 索引 + parentId 映射 |

### 练习优先级

**必会（10 分钟内写出）**：Promise 串行+重试 → 模板编译 → class 转 ES5 → 异步调度器 → EventEmitter → 深拷贝 → 防抖节流 → call/apply/bind
