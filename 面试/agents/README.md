# 面试多Agent教练系统

> 基于 GLM 智能体的面试训练闭环：训练 → 模拟 → 复盘
> 所有 Agent 共享 `weakness-tracker.md` 作为记忆文件

## 快速搭建（10分钟）

### Step 1：在 GLM 清言创建 3 个智能体

打开 [chatglm.cn](https://chatglm.cn) → 左侧「智能体」→ 创建智能体

| 智能体名称 | 用途 | 提示词文件 |
|------------|------|-----------|
| 面试教练-训练 | 知识训练、项目深挖、算法口述 | `01-训练教练.md` |
| 面试教练-模拟 | 模拟面试、评分、弱项暴露 | `02-模拟面试官.md` |
| 面试教练-复盘 | 面后分析、失分诊断、改进计划 | `03-复盘分析师.md` |

每个智能体的「系统提示词」栏复制对应文件里 ```text ``` 代码块中的内容。

### Step 2：上传知识库文件

每个智能体都要上传 `weakness-tracker.md`，再加上各自的专用文件：

**训练教练** 知识库：
- `weakness-tracker.md`（必须，每次更新后重新上传）
- `答案卡.md`
- `急救包.md`
- `题库详解.md`
- `简历/v2.md`

**模拟面试官** 知识库：
- `weakness-tracker.md`（必须，每次更新后重新上传）
- `答案卡.md`
- `简历/v2.md`
- `速查手册.md`
- `系统设计白板.md`

**复盘分析师** 知识库：
- `weakness-tracker.md`（必须，每次更新后重新上传）
- `答案卡.md`
- `面后复盘.md`
- `简历/v2.md`
- `急救包.md`

### Step 3：日常使用

```
每日循环：

上午：训练教练
├── 打开"面试教练-训练"智能体
├── 说"今天练什么" → AI 根据弱项安排训练
├── 练 4-6 题后说"更新追踪"
└── 把 AI 输出的更新片段替换到本地 weakness-tracker.md

下午：模拟面试官（隔天一次）
├── 打开"面试教练-模拟"智能体
├── 说"开始模拟：字节" → 完整模拟面试
├── 结束后 AI 输出评分报告
└── 把弱项更新片段替换到本地 weakness-tracker.md

面试后：复盘分析师
├── 打开"面试教练-复盘"智能体
├── 说"复盘面试：字节 二面"
├── 粘贴面试问题和我记得的回答
├── AI 输出失分分析 + 改进计划 + 答案卡
└── 更新本地 weakness-tracker.md
```

## 关键操作

### 更新 weakness-tracker.md（每次训练/模拟/复盘后必做）

1. AI 会输出一段 markdown 片段（表格行）
2. 复制这段内容，替换本地 `weakness-tracker.md` 中对应的行
3. 重新上传到 3 个智能体的知识库（这样下次会话 AI 能读到最新状态）

> 这是闭环的核心：训练教练发现问题 → 模拟面试官验证 → 复盘分析师诊断 → 更新追踪表 → 下次训练针对性补强

### 录音复盘流程

1. 面试时手机录音（讯飞/系统录音机）
2. 面试后讯飞听见转文字
3. 打开"面试教练-复盘"智能体
4. 说"复盘录音"并粘贴转写文本
5. AI 自动分析失分点
