# 每日定投参考 · 美股&AI基金提示词

> 使用方法：每个交易日早上（美股收盘后/开盘前），复制下方分隔线内的全部内容粘贴到 AI 对话框。

---

## 配合工具：daily_stock_analysis

**推荐部署方案**：[daily_stock_analysis](https://github.com/ZhuLinsen/daily_stock_analysis) 是一个 LLM 驱动的全自动股票分析系统，通过 GitHub Actions 零成本部署，每天自动推送分析报告到微信/飞书/Telegram。

### 部署步骤

1. **Fork 仓库** → `https://github.com/ZhuLinsen/daily_stock_analysis`
2. **配置 Watchlist**：在 GitHub repo Settings → Secrets 中设置：
   ```
   STOCK_LIST=QQQ,NVDA,MSFT,GOOGL,META,AMZN,AMD,AVGO,SOXX
   ```
3. **选择推送渠道**（推荐 Telegram 或飞书）：
   - Telegram：设置 `TELEGRAM_BOT_TOKEN` + `TELEGRAM_CHAT_ID`
   - 飞书：设置 `FEISHU_WEBHOOK_URL`
4. **GitHub Actions 会自动运行**：每天美股收盘后自动分析并推送

### Watchlist 说明

| 代码 | 用途 |
|------|------|
| QQQ | 纳斯达克 100 ETF，纳指定投的核心参考 |
| NVDA | AI 算力龙头 |
| MSFT / GOOGL / META / AMZN | AI 平台型巨头 |
| AMD / AVGO | 半导体/AI 芯片 |
| SOXX | 半导体 ETF，AI 硬件景气度指标 |

### 使用流程

```
每日早上 →
  1. 查看 daily_stock_analysis 推送的分析报告（个股技术面+基本面）
  2. 将推送内容 + 下方提示词一起粘贴给 AI
  3. AI 综合 two sources 产出「定投参考」
```

## 使用说明

### 方案一：完整版（推荐）

1. 部署 [daily_stock_analysis](https://github.com/ZhuLinsen/daily_stock_analysis)，配置 Watchlist
2. 每天早上收到推送后，将推送内容 + 提示词一起粘贴给 AI
3. AI 综合产出定投参考

优点：技术面数据由工具自动采集，准确且省时；AI 专注补充分析和定投决策

### 方案二：纯提示词版（不部署工具）

直接使用提示词，但把「需要补充搜索的信息」扩展为包含所有数据搜索：

在提示词的「需要补充搜索的信息」部分前，增加：

```
**美股大盘**
1. 搜 "Nasdaq S&P 500 Dow Jones today close {今天日期}" — 三大指数收盘数据
2. 搜 "US stock market today recap {今天日期}" — 市场复盘

**AI/科技核心标的**
3. 搜 "NVDA MSFT GOOGL META AMZN stock today" — AI 龙头股走势
4. 搜 "semiconductor AI stock today AMD AVGO" — 半导体/AI 芯片
```

### 节奏说明

- **频率**：每个交易日早上（建议 9:00-10:00，美股已收盘，A股将开盘）
- **耗时**：方案一约 3 分钟（工具已推送数据），方案二约 5 分钟（全量搜索）
- **非交易日**：周末和美股假日自动跳过
- **如果想深入某只个股**：追问"展开分析 NVDA 今天的走势和财报预期"
- **如果想调整定投标的**：修改「我的持仓情况」一节
