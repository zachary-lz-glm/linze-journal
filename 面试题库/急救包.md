## 一、八股口语化速记（5 题，每题 3 句话）

### C16. React 18 并发 + Fiber

1. **Fiber 本质是把虚拟 DOM 从树变成链表**（child/sibling/return），这样渲染可以随时中断和恢复
2. 之前是一递归到底不可中断，**现在通过 Scheduler 用 MessageChannel 按优先级调度**，高优先级任务（比如用户输入）可以打断低优先级渲染
3. React 18 新增 useTransition 标记低优先级更新、useDeferredValue 延迟渲染、**Automatic Batching 自动合并多次 setState**，不管在什么上下文里都只触发一次渲染

### C17. useEffect vs useLayoutEffect

1. **useLayoutEffect 在浏览器绘制前同步执行，会阻塞渲染；useEffect 在绘制后异步执行，不阻塞**
2. 执行顺序是：DOM 更新 → useLayoutEffect → 浏览器绘制 → useEffect
3. 所以**需要读 DOM 布局或做同步修改用 useLayoutEffect**（防止闪烁），其他都用 useEffect

### C18. 浏览器渲染管线

1. **HTML 解析成 DOM，CSS 解析成 CSSOM，合并成 Render Tree**
2. 然后 Layout 算几何位置 → Paint 画像素 → Composite 合成图层
3. **改 width/height 触发 Reflow（重排最贵），改 color 触发 Repaint，只改 transform/opacity 只触发 Composite 性能最好**

### C19. HTTP 缓存

1. **强缓存不发请求**，通过 Cache-Control 的 max-age 直接用本地缓存
2. **协商缓存发请求但可能 304 不下载内容**，用 ETag/If-None-Match 做精确比对
3. 实战配置口诀：**HTML 用 no-cache 每次协商、带 hash 的静态资源 max-age 设一年、API 用 no-store 不缓存**

### C20. TypeScript 高级类型

1. **infer 是在条件类型里"推断"一个子类型**，比如从函数类型里推断返回值类型
2. 条件类型就是类型层面的三元表达式 `T extends X ? A : B`
3. 映射类型可以遍历联合类型生成新类型，**Pick、Omit、Partial 本质都是映射类型**

## 二、算法万能模板（5 个套路，覆盖 10 题）

### 模板 1：哈希表（万能第一步）

> 适用：两数之和、字母异位词分组、多数元素、重复检测

```
套路：遇到"找配对/找重复/统计次数"→ 开 Map
口诀：遍历时查 Map，查到就返回，查不到就存进去

function twoSum(nums, target) {
    const map = new Map();
    for (let i = 0; i < nums.length; i++) {
        if (map.has(target - nums[i])) return [map.get(target - nums[i]), i];
        map.set(nums[i], i);
    }
}
```

### 模板 2：双指针（数组/链表通用）

> 适用：三数之和、反转链表、最长子串、合并有序数组

```
套路 A — 对撞指针：排序后左右两端往中间夹
function threeSum(nums) {
    nums.sort((a, b) => a - b);
    for (let i = 0; i < nums.length - 2; i++) {
        let left = i + 1, right = nums.length - 1;
        while (left < right) {
            const sum = nums[i] + nums[left] + nums[right];
            if (sum === 0) { /* 收集结果, 跳过重复 */ }
            else if (sum < 0) left++;
            else right--;
        }
    }
}

套路 B — 快慢指针：链表找中点/环/倒数第N个
```

### 模板 3：BFS 队列（层序/最短路径/岛屿）

> 适用：二叉树层序遍历、岛屿数量、单词接龙

```
套路：队列 + while 循环 + 逐层处理
function levelOrder(root) {
    if (!root) return [];
    const queue = [root], result = [];
    while (queue.length) {
        const level = [], size = queue.length;
        for (let i = 0; i < size; i++) {
            const node = queue.shift();
            level.push(node.val);
            if (node.left) queue.push(node.left);
            if (node.right) queue.push(node.right);
        }
        result.push(level);
    }
    return result;
}

岛屿数量：BFS/DFS 遇到 1 就染色成 0，计数 +1
```

### 模板 4：滑动窗口（子串/子数组问题）

> 适用：无重复最长子串、最小覆盖子串、长度最小的子数组

```
套路：左右指针维护窗口，右扩左缩
function lengthOfLongestSubstring(s) {
    const set = new Set();
    let left = 0, max = 0;
    for (let right = 0; right < s.length; right++) {
        while (set.has(s[right])) { set.delete(s[left++]); }
        set.add(s[right]);
        max = Math.max(max, right - left + 1);
    }
    return max;
}

口诀：右指针一直走，遇到重复就缩左
```

### 模板 5：DFS 回溯（排列/组合/子集）

> 适用：全排列、组合总和、子集、N皇后

```
套路：选 → 递归 → 撤销选择
function permute(nums) {
    const result = [], used = new Array(nums.length).fill(false);
    function backtrack(path) {
        if (path.length === nums.length) { result.push([...path]); return; }
        for (let i = 0; i < nums.length; i++) {
            if (used[i]) continue;
            used[i] = true;
            path.push(nums[i]);
            backtrack(path);
            path.pop();
            used[i] = false;
        }
    }
    backtrack([]);
    return result;
}

口诀：选了标记 → 往下走 → 回来撤销标记
```

### 补充模板 6：二叉树 DFS 路径类（字节第 3、4 名）

> 适用：求根到叶子数字之和、路径总和、二叉树所有路径

```
套路：递归走到叶子，累加路径上的值
口诀：带参数往下传，到叶子收结果

// LC112 路径总和
function hasPathSum(root, targetSum) {
    if (!root) return false;
    if (!root.left && !root.right) return root.val === targetSum;
    return hasPathSum(root.left, targetSum - root.val)
        || hasPathSum(root.right, targetSum - root.val);
}

// LC129 求根到叶子节点数字之和
function sumNumbers(root) {
    function dfs(node, sum) {
        if (!node) return 0;
        sum = sum * 10 + node.val;
        if (!node.left && !node.right) return sum;
        return dfs(node.left, sum) + dfs(node.right, sum);
    }
    return dfs(root, 0);
}
```

### 补充模板 7：合并两个有序数组（字节第 2 名，三家通吃）

```
function merge(nums1, m, nums2, n) {
    let i = m - 1, j = n - 1, k = m + n - 1;
    while (j >= 0) {
        nums1[k--] = (i >= 0 && nums1[i] > nums2[j]) ? nums1[i--] : nums2[j--];
    }
}

口诀：从后往前，不用额外空间
```

### 补充模板 8：字符串模拟（字节高频）

```
// LC165 比较版本号（字节第 7 名，11 次）
function compareVersion(a, b) {
    const aa = a.split('.'), bb = b.split('.');
    const len = Math.max(aa.length, bb.length);
    for (let i = 0; i < len; i++) {
        const x = parseInt(aa[i] || 0), y = parseInt(bb[i] || 0);
        if (x > y) return 1;
        if (x < y) return -1;
    }
    return 0;
}

// LC415 字符串相加（字节 8 次）
function addStrings(a, b) {
    let i = a.length - 1, j = b.length - 1, carry = 0, res = '';
    while (i >= 0 || j >= 0 || carry) {
        const sum = (i >= 0 ? +a[i--] : 0) + (j >= 0 ? +b[j--] : 0) + carry;
        res = (sum % 10) + res;
        carry = Math.floor(sum / 10);
    }
    return res;
}

口诀：按位处理，从后往前，注意进位
```

### 补充 9：最大子序和（字节第 5 名，DP 入门）

```
function maxSubArray(nums) {
    let max = nums[0], cur = nums[0];
    for (let i = 1; i < nums.length; i++) {
        cur = Math.max(nums[i], cur + nums[i]);
        max = Math.max(max, cur);
    }
    return max;
}

口诀：要么接前面的，要么从自己重新开始
```

### 补充 10：手撕快速排序（美团第 1 名）

```
function quickSort(arr) {
    if (arr.length <= 1) return arr;
    const pivot = arr[0];
    const left = arr.slice(1).filter(x => x <= pivot);
    const right = arr.slice(1).filter(x => x > pivot);
    return [...quickSort(left), pivot, ...quickSort(right)];
}

// 原地版（面试官可能要求）
function quickSortInPlace(arr, lo = 0, hi = arr.length - 1) {
    if (lo >= hi) return;
    let i = lo, j = hi, pivot = arr[lo];
    while (i < j) {
        while (i < j && arr[j] >= pivot) j--;
        arr[i] = arr[j];
        while (i < j && arr[i] <= pivot) i++;
        arr[j] = arr[i];
    }
    arr[i] = pivot;
    quickSortInPlace(arr, lo, i - 1);
    quickSortInPlace(arr, i + 1, hi);
}
```
