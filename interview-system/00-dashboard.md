> 用法：每天只看这一页。它负责告诉你今天练什么、不要练什么、产出什么。

## 当前定位

**高级前端 / AI 工程型候选人**

主线：

```text
Schema 生成 -> Schema 渲染 -> benefit SDK 同构复用 -> prd-tools AI 工程化
```

不要把自己讲成“纯页面开发”。你的职级杠杆是：

- 做过复杂中台的 Schema 架构
- 懂 BFF 编排和前后端联动边界
- 做过组件库工程化和 Monorepo 治理
- 能把 AI 从 prompt 玩具落到证据链、质量门控和反馈回流

## 72 小时冲刺节奏

### D-3 / D-2

1. 30 秒开场录音 3 遍。
2. 选一家公司做 45 分钟 mock。
3. 刷 3 道算法，每题必须讲复杂度。
4. 把 mock 崩掉的问题补进 `03-answer-cards.md`。

### D-1

1. 只看 `03-answer-cards.md`。
2. 三家公司各做 20 分钟 mock。
3. 画 Schema、prd-tools、benefit SDK 三张架构图。
4. 睡前听自己的开场录音，不再扩资料。

```text
兜底：实际遇到过 -> 核心矛盾 -> 具体做法 -> 如果重做
```
